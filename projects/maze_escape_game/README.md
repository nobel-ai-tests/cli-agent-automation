# Maze Escape Game

A polished top-down maze escape game built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Multiple Levels**: Distinct maze layouts with progression.
- **Persistence**: Best scores (minimum steps) for each level are saved in local storage.
- **Controls**: 
    - **Keyboard**: Arrow Keys or WASD.
    - **Mobile/Touch**: Large on-screen directional buttons.
- **Responsive Design**: The maze scales automatically for smaller screens.
- **Collision Detection**: Reliable wall collision logic.
- **Win Condition**: Reach the green exit ('E') to advance.

## How to Play
1. Use the controls to move the red circle (Player).
2. Navigate through the path without hitting the dark walls.
3. Reach the green exit to complete the level.
4. If you get stuck, use the **Reset Game** button.

## Implementation Details
- **Grid Layout**: Built using CSS Grid and dynamic column generation.
- **Styling**: Modern look with shadows, rounded corners, and animations.
- **Vanilla JS**: No external dependencies or frameworks used.
