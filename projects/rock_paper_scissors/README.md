# Rock, Paper, Scissors Game

A clean, responsive, web-based Rock, Paper, Scissors game built with vanilla HTML, CSS, and JavaScript.

## Features

- **Interactive UI:** Large, easy-to-click buttons with visual feedback on hover and selection.
- **Score Tracking:** Real-time scoreboard for both Player and Computer.
- **Visual Feedback:** Animations and color coding for wins (green), losses (red), and draws (yellow).
- **Responsive Design:** Works beautifully on desktop and mobile browsers.
- **Reset Functionality:** Easily restart the game and clear all scores.

## How to Play

1. Choose one of the three options: **Rock**, **Paper**, or **Scissors**.
2. The computer will randomly make its choice.
3. The winner of the round is determined based on:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Your score and the computer's score will update automatically.
5. Click **Reset Game** to start over from zero.

## Project Structure

- `index.html`: The structure of the game UI.
- `style.css`: The styling and animations for a polished look.
- `script.js`: The game logic and DOM manipulation.

## Implementation Details

- **No external dependencies:** Built entirely with standard web technologies.
- **Accessible emojis:** Uses standard emojis for a fun, visual experience.
- **State Management:** Tracks scores and game state locally in the session.
