# Typing Speed Game

A simple, polished typing speed game built with HTML, CSS, and JavaScript.

## Features
- **Random Text**: Displays random sentences for the player to type.
- **Real-time Feedback**: Highlights correct and incorrect characters as you type.
- **Live Metrics**: Tracks Words Per Minute (WPM) and Accuracy in real-time.
- **Fixed Duration**: The game runs for a set time (e.g., 60 seconds).
- **Results & Restart**: Displays final statistics at the end and allows the player to start a new round.

## Implementation Plan
1. **HTML Structure**: Create a clean UI with a container for the target text, an input field, a timer display, and a results modal. (Completed)
2. **Styling (CSS)**: Use a modern, responsive design with CSS Flexbox/Grid. Add visual cues for typing progress. (Completed)
3. **Logic (JavaScript)**:
    - Manage a collection of sentences. (Enhanced)
    - Handle keyboard input and compare it to the target text. (Completed)
    - Calculate WPM and Accuracy. (Refined for better precision)
    - Implement a countdown timer. (Completed)
    - Handle game state (Start, Running, Finished, Restart). (Completed)
4. **Polish**: Add more varied sentences and improve the visual feedback for mistakes.

## How to Run
Open `index.html` in any modern web browser.
