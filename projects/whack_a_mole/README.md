# Whack-a-Mole Game

A simple browser-based Whack-a-Mole game built with vanilla HTML, CSS, and JavaScript.

## Completed Features

- [x] **HTML Structure**: Grid-based game board with score and timer displays.
- [x] **Styling**: CSS Grid for layout, responsive design, and animations for mole movement.
- [x] **Game Logic**:
    - Randomized mole appearing and disappearing.
    - Score tracking with whack feedback.
    - 30-second countdown timer.
    - Start/Restart functionality with clean state reset.
- [x] **UI Enhancements**:
    - Game Over overlay with final score.
    - Improved mole visuals (eyes, nose, whacked state).
    - Responsive grid layout.
- [x] **Assets**: All visuals are created using pure CSS (no external images required).

## How to Play

1. Open `index.html` in any modern web browser.
2. Click the "Start Game" button.
3. Click on the moles as they pop up to increase your score.
4. The game ends after 30 seconds, and your final score will be displayed.
